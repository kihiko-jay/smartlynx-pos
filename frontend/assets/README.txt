Place your icon files here before building:

  icon.ico  — Windows icon (256x256 recommended)
  icon.png  — Linux icon (512x512 recommended)  
  icon.icns — macOS icon (optional)

You can create a free icon at:
  https://icon.kitchen  (generates all formats from one image)
  https://realfavicongenerator.net

For a quick placeholder during development, any .ico file will work.
Download a free one from: https://www.flaticon.com
